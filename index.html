Fed-Avis: A Lightweight Federated Learning Assisted Anti-Vehicle Infiltration System

FedAvis is a federated learning-based system designed to enhance military security using federated learning, lightweight models (TinyYOLOv4, MobileNetV2), and edge devices like Raspberry Pi. Trained on 6,772 vehicle images, it achieves 6.5-second average alert latency via Azure service bus. It improves efficiency, accuracy, privacy, and energy use over traditional systems.
